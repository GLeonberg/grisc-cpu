# Grisc CPU
A Custom Risc CPU

Features:    

- Built in UART interface
- 64x64 VGA Graphics Output
- 16-bit RISC Architecture  
    - 16 bit words  
    - 32 bit instructions     
    - Load/Store  
    - Harvard Architecture   

Also Includes: (Software Toolchain)  
- gemu: An emulator written in Python
- gasm: An assembler written in Python
